Quick Note on Jupyter Notebooks

Quick note on the Windows installation video-- when I'm running cells in Jupyter Notebooks, I'm clicking Shift+Enter to run the cell with print "Hello World". (We'll learn about a lot more about the Jupyter Notebooks in Lecture 10, but this is just a heads up for the installation videos).

Also for the Windows Installation, if for some reason you can't find the Anaconda Command Prompt mentioned in the videos, you can also search for cmd or powershell in windows and use those options as your command prompt instead.
