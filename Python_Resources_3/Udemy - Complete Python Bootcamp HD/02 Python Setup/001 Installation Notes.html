Installation Notes

COURSE MATERIALS INSTALLATION GUIDE

In this text lecture we'll go over a written guide to installing the course materials. If you prefer to follow a video guide, check out the next lectures that correspond to your OS, however, installation is actually very straight-forward.

COURSE INSTALLATION STEPS

  1. Go to the Jupyter Website (jupyter.org)
  2. Scroll down and and click the Install the Notebook button
  3. The button will take you to the installation documentation.
  4. Follow the instructions there that best suit your set-up, but I recommend following the instructions for downloading Jupyter and Python with the Anaconda Distribution.
  5. Go to the Anaconda Installation page and click on the Graphical Installer for your system to download the installer.
  6. Follow the directions for the graphical installer you chose.
  7. Once you've successfully downloaded Anaconda you can begin with the installation commands for Jupyter. All installation commands should be run in the Terminal (for Mac and Linux) or the Command Prompt/Powershell (Windows). (Mac users should just search for terminal in Spotlight search, Windows Users just search ofr either powershell or cmd in your windows search tool to find your approriate installation tool). You also have the option of using the Anaconda Command Prompt as shown in the videos.
  8. Go to your terminal/command prompt and type conda install jupyter
  9. You should see some text show up describing the installation.
  10. Once the installation is done, in your terminal/command prompt type: jupyter notebook
  11. You should eventually see a new tab open up in your browser for you to begin using Jupyter Notebooks.
  12. For more information on how to use the Jupyter Notebooks, refer to the other lectures in this section.
  13. For more information on the Jupyter Notebook system in general, check out the official documentation.
